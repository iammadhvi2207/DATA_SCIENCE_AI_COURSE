## 📌 About the Course

This repository contains all course materials, assignments, notes, practice exercises, and projects completed during the **Data Science & Artificial Intelligence Program at GRASTech**.

The course is designed to provide a strong foundation in data analysis, machine learning, deep learning, and AI-driven solutions through hands-on projects and real-world applications. It covers the complete workflow from data collection and preprocessing to model development, evaluation, and deployment.
